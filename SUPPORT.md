# Support

Use GitHub Issues for bug reports, installation problems, feature requests, and reproducible examples.

When reporting a problem, include:

- TranscriptoScope version.
- Windows version.
- R version, if known.
- Whether the input type was read counts, normalized expression, fold changes with adjusted p-values, or fold changes only.
- A small example file that reproduces the issue, if the data can be shared publicly.
- The exact error message or screenshot.

Please do not attach private, protected, unpublished, or regulated datasets to public issues.
